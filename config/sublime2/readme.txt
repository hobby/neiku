
一、说明
commands.py 是sublime text 2的中sftp插件的破解版文件，破解内容：当未注册sftp插件时，每10次上传操作都会有一个弹窗，影响使用，这个破解把弹窗去掉了。

二、安装方法
（1）将commands.py文件复制粘贴到：<系统所在分区盘符（例如C）>:\Users\<用户名>\AppData\Roaming\Sublime Text 2\Packages\SFTP\sftp
（2）重启sublime
（3）忽略相同目录下面的commands.pyc
